# React-TypeScript

Minimal boilerplate for a single-page app using Mobservable, Mobservable-React, React, TypeScript 1.6 with TSX, and Visual Studio Code.

Initial install:

* Install Node.js
* `npm install -g tsd`
* `npm install`
* `tsd install`

Build:

* `npm run build`

To view the app:

* `npm start`
